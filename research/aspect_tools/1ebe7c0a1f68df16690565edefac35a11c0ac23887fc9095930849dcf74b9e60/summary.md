# Summary

Daytona pivoted in February 2025 from being a developer-environment platform to being secure elastic infrastructure for running AI-generated code. Sandboxes spin up in 90ms, support stateful operations, multiple languages (Python, TypeScript, Go), full file system / process / git operations, and can be hosted, self-hosted, or run hybrid where Daytona orchestrates and the user owns execution.

For dark factories Daytona is the explicit sandbox choice of LangChain's Open SWE reference architecture, and is widely used alongside or instead of e2b. Its API-first design ("every operation accessible to an AI Agent") and self-hostable open-source stack make it especially attractive to enterprise dark-factory builds that cannot ship code to a hosted vendor.

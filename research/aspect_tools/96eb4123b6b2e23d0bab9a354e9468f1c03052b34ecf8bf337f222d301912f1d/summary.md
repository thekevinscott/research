# Summary

Factory.ai's "Factory is GA" announcement marks the general availability of an enterprise-grade agent-native platform whose autonomous "Droids" handle complete SDLC tasks - coding, testing, PR creation, deployment QA, and SRE-style incident response - across IDEs, web, CLI, Slack, and Linear. Specialized Droids (Code, Knowledge/Tech-Writing, Reliability) cover different stages of the lifecycle.

For dark factories Factory is one of the most directly applicable commercial offerings: it is model- and IDE-agnostic, certified for ISO 42001, SOC 2, ISO 27001, GDPR, CCPA, and already deployed at MongoDB, Zapier, Framer, and Clari. The Code Droid technical report describes exactly the seed -> harness -> feedback loop pattern, making Factory one of the heaviest commercial enabler in real production dark-factory deployments.

Overview:

Persistence:

> "A Cloudflare Sandbox is a persistent, isolated environment powered by Cloudflare Containers, giving AI agents a persistent, isolated environment: a real computer with a shell, a filesystem, and background processes that starts on demand and picks up exactly where it left off."

Capabilities:

> "A Sandbox today is a full development environment: a terminal you can connect a browser to, a code interpreter with persistent state, background processes with live preview URLs, a filesystem that emits change events in real time, egress proxies for secure credential injection, and a snapshot mechanism that makes warm starts nearly instant."

Execution:

> "With Sandbox, you can execute Python scripts, run Node.js applications, analyze data, compile code and more. The SDK provides methods for executing commands, cloning repositories, writing files, and managing processes through a TypeScript API."

Dynamic Workers:

> "Cloudflare also introduced Dynamic Workers, which allow execution of AI-generated code in secure, lightweight isolates that are 100 times faster than traditional containers, enabling millisecond startup times for AI agent sandboxing."

## Relevance to dark factories

Cloudflare Sandboxes give agents a persistent computer at the edge, with snapshot warm-starts and credential-injecting egress proxies - which directly addresses the dark-factory need for secure, fast, persistent agent state. Dynamic Workers' millisecond isolates push the cold-start problem out of the loop.

# Summary

Claude Code is Anthropic's terminal-first agentic coding tool, launched Feb 2025 and now (2026) one of the most widely adopted AI coding tools with 101k GitHub stars. It operates fully autonomously: reading the full codebase, planning across files, editing, running tests, debugging, and iterating. It is powered by Claude Opus 4.6 (long-horizon) and Sonnet 4.6 (default) with 1M-token contexts.

For dark factories, Claude Code is a leading "worker" agent. Its Agent Teams feature lets a lead Claude coordinate multiple Claude Code instances on subtasks - a built-in dark-factory swarm topology. It is the most common substrate for orchestration platforms (Ruflo, Coder, Docker Sandboxes) and observability platforms (Datadog MCP, Helicone, LangSmith) that wrap dark-factory pipelines.

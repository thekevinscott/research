# Summary

Kimi K2 (Moonshot AI) is a 1T-parameter MoE model with 32B active parameters tuned specifically for agentic coding. The 2026 K2.6 release adds native multimodality and an Agent Swarm architecture that horizontally scales to 300 sub-agents executing 4,000 coordinated steps in a single autonomous run, with explicit support for persistent 24/7 background agents.

For dark factories Kimi K2 is the leading open-weights worker option: it is engineered for the long-horizon, swarm-orchestrated, run-without-human-oversight regime that defines the pattern. Available via Microsoft Foundry, Moonshot's own Kimi API, and Hugging Face for self-hosting, it provides a credible non-Western alternative for organizations that need on-prem or sovereign-cloud dark factories.

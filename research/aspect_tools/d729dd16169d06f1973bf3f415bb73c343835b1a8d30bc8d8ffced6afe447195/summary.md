# Summary

Cursor automatically manages git worktrees so multiple agents can work in parallel without conflicts, and reports that 30% of its own PRs are now agent-authored.

The April 2026 Cursor 3 release - and the new TypeScript SDK for programmatic coding agents with sandboxed cloud VMs, subagents, hooks, and token pricing - turns Cursor into a substrate that other teams can use to build their own dark factories. With self-hosted Cloud Agents now available, this is one of the most enterprise-ready packaged dark-factory stacks in 2026.

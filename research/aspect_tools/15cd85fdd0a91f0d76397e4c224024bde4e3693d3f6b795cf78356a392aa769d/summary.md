# Summary

Aider is the most popular open-source AI pair-programming tool (Apache 2.0, 41.6k GitHub stars, 5.3M+ PyPI installs). It runs in the terminal, builds a repo map for context, supports virtually any LLM (Claude, GPT-4o, DeepSeek, local models), and auto-commits changes with sensible messages so users can diff and undo via git.

For dark factories Aider is the canonical "build your own" worker: free, scriptable, model-agnostic, and trivially embeddable in CI. Teams who want the seed -> code -> commit loop without a vendor lock to Cognition/Cursor/Anthropic frequently start with Aider plus an external harness layer.

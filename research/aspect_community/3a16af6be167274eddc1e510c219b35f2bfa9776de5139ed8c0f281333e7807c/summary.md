# Summary

BCG Platinion's "Dark Software Factory" report (March 2026) is the most cited consultancy-side framing of the pattern. It defines the dark factory as an autonomous software delivery pipeline, holds up Spotify (650 AI PRs/month, no human-written code since December 2025) and StrongDM (three engineers, no human code) as case studies, and reports BCG's own five-day enterprise application conversion as a productivity proof point.

The report's strategic argument — that quality depends jointly on how the factory is architected and how precisely the organisation can articulate intent — is now the standard executive framing. It moved the conversation from "can this work?" to "where do we deploy it?" in C-suite circles, and is the principal channel by which the pattern reached non-technical leadership.

# GitHub — ComposioHQ/agent-orchestrator README

Agent-orchestrator (Composio) is an open-source orchestrator that spawns multiple Claude Code, Codex, Aider, or OpenCode agents in parallel — each in an isolated git worktree on its own branch — and autonomously handles CI fixes, merge conflicts, and code review responses. It includes a coordination layer, dashboard, and CI integration. Reportedly 6.4k GitHub stars by mid-2026.

Positioned as the practical glue for Level 4–5 workflows where one developer drives many agents at once. The README emphasizes autonomous PR lifecycle handling — agents fix CI failures and respond to review comments without human approval at each step. This is one of the few public, reusable orchestrators and is cited heavily in 2026 "open-source orchestrator" roundups.

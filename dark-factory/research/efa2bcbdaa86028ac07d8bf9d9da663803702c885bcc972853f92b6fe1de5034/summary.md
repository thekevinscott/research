# RooCodeInc/Roo-Code

Roo Code (formerly Roo Cline) is a VS Code coding agent forked from Cline that adds custom 'modes' (Architect, Debug, Test, Code, Ask) to constrain what the agent can do at each step, plus diff-based editing that cuts ~30% of token cost vs Cline. It is Apache-2.0 licensed and ships a JetBrains bridge.

Roo Code occupies the same pipeline slot as Cline — IDE-bound coding-agent harness — but its mode system is a near-direct mirror of the Dark Factory idea that an agent's role should be narrowed by spec. In a Dark Factory build, Roo's modes are the closest open analog to the multi-phase nodes in StrongDM's Attractor graph.

# Skyvern-AI/skyvern

Skyvern automates browser-based workflows using LLMs and computer vision; the agent reads a web page, plans a sequence of actions, and executes them in a real browser. Designed for end-to-end task automation across third-party services without bespoke API integrations.

Skyvern is in the same DTU-adjacent niche as browser-use — it lets an agent operate third-party services through their UIs. Less broadly adopted than browser-use but with a heavier vision-model emphasis; in Dark Factory pipelines it is an alternative to behavioral-clone DTUs for low-volume long-tail services.

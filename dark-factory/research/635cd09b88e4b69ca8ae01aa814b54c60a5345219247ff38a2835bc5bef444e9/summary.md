# StrongDM — Software Factory products index

Index page enumerating the three components StrongDM open-sourced as "useful components from our Software Factory": CXDB (AI context store), StrongDM ID (federated identity for humans, workloads, and AI agents with path-scoped sharing), and Attractor (non-interactive coding agent).

The page is short but useful as the canonical map of StrongDM's tools-layer contributions to the Dark Factory pattern. It establishes that the factory is a product family, not a monolith, and makes identity (StrongDM ID) a first-class component alongside the agent and the context store.

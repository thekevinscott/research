# GitHub Blog — Copilot coding agent announcement

GitHub's announcement of the Copilot coding agent: an asynchronous autonomous developer that can be assigned a GitHub issue and work in the background — writing code, running tests, opening a PR — without further human input. Generally available for all paid Copilot subscribers as of mid-2026.

GitHub also announced Agent HQ at Universe 2025: a control plane orchestrating agents from Anthropic, OpenAI, Google, Cognition, and xAI across GitHub, VS Code, CLI, and Mobile under a single Copilot subscription. This positions GitHub as the multi-vendor orchestration layer rather than a single-agent vendor — a key piece of the tools landscape for Level 4–5 operations.

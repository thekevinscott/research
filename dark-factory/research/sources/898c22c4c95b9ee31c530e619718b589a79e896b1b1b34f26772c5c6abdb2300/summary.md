# promptfoo/promptfoo

Promptfoo is a CLI and library for evaluating, comparing, and red-teaming LLM apps and agents — declarative YAML configs, CI/CD integration, used by OpenAI and Anthropic per its README. OpenAI acquired Promptfoo on March 9 2026 for $86M; the OSS repo continues.

Promptfoo is a general LLM eval harness with first-class support for agents and security probing. In Dark Factory pipelines it is the deterministic-checks layer (regex, JSON-schema, tool-call shape, latency budgets) that gates each scenario before LLM-judge passes are spent.

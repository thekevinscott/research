# StrongDM — CXDB product page

CXDB is StrongDM's self-hosted AI Context Store: a content-addressed Turn DAG plus blob CAS for storing every turn of every agent conversation. It supports branching from any turn (O(1) forks), content deduplication via BLAKE3 hashing, type-safe projections for UIs, and a visual debugger.

Marketed as the immutable substrate underneath a Software Factory, CXDB is positioned as the production-grade equivalent of agent memory — letting teams replay, branch, and audit autonomous coding sessions. Released open-source alongside Attractor and StrongDM ID as components of the StrongDM Software Factory.

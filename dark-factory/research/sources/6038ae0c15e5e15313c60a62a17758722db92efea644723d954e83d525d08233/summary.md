# Augment Code — 9 Open-Source Agent Orchestrators (2026)

Augment Code's 2026 roundup of nine open-source agent orchestrators for AI coding. The list functions as a tools-layer map: agent-orchestrator (Composio) for parallel Claude Code sessions on git worktrees, Ralphify for autonomous Ralph loops, IttyBitty for multi-agent Claude Code, Aider's loop wrappers, plus several niche orchestrators.

The piece argues orchestrators — not single agents — are the bottleneck for Dark Factory operation: above 4–8 concurrent agents per developer, the limit becomes review and merge orchestration, not code generation. It scores each entry on autonomy level, parallelism, and PR lifecycle handling. Useful synthesis source for the tools-aspect map.

# browserbase/stagehand

Stagehand is Browserbase's SDK for browser agents: combines natural-language commands ('act', 'extract', 'observe') with code-level Playwright control. Pitched as the production-grade middle ground between brittle scripts and free-form LLM agents.

Stagehand fits the same DTU-substitute slot as browser-use and Skyvern, but is the most popular choice when teams want determinism — Playwright underneath, LLM on top. In Dark Factory pipelines Stagehand is the third-party-service interaction layer that scenario tests can replay reliably.

# generalaction/emdash

Emdash (YC W26) is a provider-agnostic desktop app that runs multiple coding agents in parallel, each isolated in its own git worktree, locally or over SSH. It supports 24 CLI agents including Claude Code, Codex, OpenCode, Gemini and Amp, branding itself as 'Open-Source Agentic Development Environment'.

Emdash is another parallel-agent orchestrator, oriented at human operators running fleets from a desktop UI. In Dark Factory terms it is the supervised-mode counterpart of agent-orchestrator: same git-worktree-per-agent architecture, but a desktop GUI rather than a CI-style dashboard. Useful as a transitional tool for teams moving from Level 4 to Level 5.

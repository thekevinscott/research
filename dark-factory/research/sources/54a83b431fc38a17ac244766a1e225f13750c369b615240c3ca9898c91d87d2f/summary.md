# OpenHands/OpenHands

OpenHands (formerly OpenDevin, originally All-Hands-AI/OpenHands) is an open platform for software-development AI agents. It provides an agent SDK, a CLI, a local GUI with a REST API, plus a hosted cloud and self-hostable enterprise option. Agents can modify code, run commands, browse the web, call APIs, and reproduce common SWE-bench-style behaviors; the project is research-grade and was published as an OpenReview paper.

OpenHands is a general-purpose coding-agent harness — the kind of harness that sits where Attractor sits in the StrongDM stack, but provider-agnostic and community-driven. In a Dark Factory pipeline it is the most mature open replacement for the proprietary inner loop. The org URL has migrated (All-Hands-AI/OpenHands now redirects to OpenHands/OpenHands); active development continued through 2026.

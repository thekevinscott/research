# GitHub — cline/cline README

Cline is an open-source autonomous coding agent that runs as a VS Code extension. Reported metrics: 61.2k GitHub stars and 5M+ installs by mid-2026. Cline can create and edit files, execute terminal commands, drive a headless browser, and call MCP tools — with every action requiring human approval before execution.

Cline is the canonical "human-in-the-loop" Level-3/4 agent in the open-source landscape. Positioned in contrast to YOLO-style autonomous agents, it represents the cautious branch of the same tooling tree. Other agents (Roo Code, OpenCode) are explicit forks. Often cited as a feeder tool for Dark Factory experimentation: teams ramp up by relaxing approval gates as harness coverage grows.

# Simon Willison — Codex CLI 0.128.0 adds /goal

Willison's link-blog summary of the Codex CLI 0.128.0 release: a /goal slash command that turns Codex into a long-horizon autonomous agent, persisting state across sessions. You give it a high-level objective; it loops plan → act → test → review until verifiably complete or paused.

The post connects /goal to the broader "ralph loop" pattern (autonomous re-prompting against a fixed objective), positioning OpenAI's release as a built-in equivalent of community frameworks like Ralphify. Willison notes the feature ships behind a feature flag and is best for tasks with a clear definition-of-done — codebase migrations, end-to-end refactors, multi-file feature builds. Dated April 30 2026.

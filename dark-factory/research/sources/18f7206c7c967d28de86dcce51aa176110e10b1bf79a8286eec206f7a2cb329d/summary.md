# Hacker News — Show HN: OctopusGarden, an autonomous software factory

The Show HN announcement (March 4 2026) for **OctopusGarden**, an open-source implementation of the Dark Factory pattern by Foundatron. Pitch: write specs and scenarios, OctopusGarden orchestrates AI coding agents to generate, build (in Docker), validate against scenarios the agent never sees, score satisfaction probabilistically (0–100, not boolean), and iterate failures back as feedback until convergence. The author explicitly cites StrongDM's holdout-scenarios and probabilistic-satisfaction patterns as inspiration, and notes the system has been validated mostly on standard CRUD/REST API apps so far.

The comment thread contains practitioner reactions — questions about scenario authoring overhead, whether the LLM judge can be reward-hacked, cost per converged build, and how the system behaves when convergence stalls. Several commenters compare it to other open-source factories (Druids, Loki Mode, the-startup) and note that 2026 is producing an open-source ecosystem around the StrongDM doctrine even though StrongDM's own Attractor ships only as a spec. Useful primary-source datapoint that the Dark Factory pattern is being actively replicated outside its originating company.

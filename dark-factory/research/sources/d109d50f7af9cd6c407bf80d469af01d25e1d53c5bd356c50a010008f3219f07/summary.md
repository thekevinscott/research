# anthropics/claude-code

Claude Code is Anthropic's official agentic CLI: terminal-resident, codebase-aware, executes routine tasks, explains complex code, handles git workflows. The plugin system supports custom slash commands, specialized sub-agents, hooks, and MCP servers configured via .claude/settings.json.

Claude Code is the most-cited closed-implementation coding-agent harness powering Dark Factory builds in 2026 — Willison's case study notes it as the kind of harness StrongDM, Cursor and others wrap. The OSS surface (plugins, agent SDK, hooks, MCP) is what makes it composable into pipelines despite the core being closed.

# eastlondoner/vibe-tools

vibe-tools (formerly cursor-tools) is a CLI that an AI coding agent can call to expand its own capabilities: web search, repo Q&A, doc generation, browser control (headless or attached), YouTube video analysis, plus instruction-file installation tailored to Cursor / Claude Code / Codex / Windsurf / Cline / Roo. It is installed globally and configures the host editor's agent ruleset on install.

vibe-tools is best understood as a tool-belt that any non-interactive coding agent can call — the same role that StrongDM's internal tools play inside Attractor. In Dark Factory pipelines it is the OSS shortcut to give an agent web/browse/search abilities without writing each tool from scratch.

# GitHub — RooCodeInc/Roo-Code README

Roo Code is an open-source AI coding extension forked from Cline that adds a multi-mode system: Code, Architect, Ask, and Debug modes. It positions itself as "a whole dev team of AI agents in your code editor" — multiple specialized roles cooperating on a single task with cloud-based autonomous variants.

Notable 2026 development: Roo Code Cloud, Router, and the original team's product line are slated to shut down May 15 2026, with a community fork carrying the project forward and Cline recommended as an alternative. The trajectory is itself a useful signal — multi-mode agent tooling is in flux as Dark Factory norms shift toward orchestrators (Composio, Agent HQ) over single-IDE plugins.

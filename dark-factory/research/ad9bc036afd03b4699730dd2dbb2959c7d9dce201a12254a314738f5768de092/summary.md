# GitHub — OpenHands/software-agent-sdk README

The OpenHands Software Agent SDK is a Python and REST-API toolkit for building software-engineering agents — MIT-licensed, designed for production deployments. The default agent is a few lines of code; advanced agents support custom tools, memory management, multi-LLM routing, sandboxed execution, lifecycle control, and built-in security analysis.

OpenHands is a top performer on SWE-bench (72% with Claude Sonnet 4.5 extended thinking), SWT-bench, and multi-SWE-bench. The repo backs the MLSys 2026 paper (arxiv 2511.03690). Adopted by AMD, Apple, Google, Amazon, Netflix, and NVIDIA as of mid-2026 and backed by an $18.8M Series A. Currently the most capable open-source coding agent stack with a paper trail.
